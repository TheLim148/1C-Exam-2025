# Билет №25
## Устные задания
**1. Опишите систему работы с графическими схемами в «1С: Предприятие».**
Графические схемы — это визуальные модели процессов и структур.
Основные элементы:
- Создаются через объект «Графическая схема».
- Используются для отображения бизнес-процессов, маршрутов, состояний.
- Позволяют размещать фигуры, линии, подписи.
- Могут быть встроены в обработки, документы или справочники.
- Используются как элемент интерфейса и аналитического инструмента.
Графические схемы помогают визуализировать структуру и логику работы системы.

**2. Объясните принципы интеграции с системами электронного документооборота.**
Интеграция с ЭДО включает:
- Использование специализированных протоколов и форматов (XML, УПД).
- Работа с операторами ЭДО через API или встроенные механизмы 1С.
- Отправку, получение и хранение юридически значимых документов.
- Сопоставление внешних документов с объектами 1С (счета, накладные, акты).
- Поддержку электронной подписи и проверку её корректности.
Интеграция позволяет автоматизировать обмен документами и уменьшить бумажный оборот.
