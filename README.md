# Билет №24
## Устные задания
**1. Опишите механизм полнотекстового поиска в «1С: Предприятие».**
Полнотекстовый поиск обеспечивает быстрый поиск данных по словам и фразам.
Основные элементы:
- Индексация текстовых полей справочников и документов.
- Поиск проводится по отдельным словам и их формам.
- Поддерживается морфология русского языка.
- Настраивается, какие объекты включать в индекс.
- Используется через стандартные команды поиска или программно.
Полнотекстовый поиск ускоряет навигацию и упрощает работу с большими объёмами данных.

---
**2. Объясните принципы оптимизации запросов и работы с индексами.**
Основные способы оптимизации:
- Использование ограничений по отбору (ГДЕ), чтобы сокращать объём выборки.
- Избегание ненужных соединений таблиц.
- Использование параметров для фильтрации.
- Ограничение объёма выбираемых данных через ПЕРВЫЕ N.
Индексация:
- Индексы ускоряют поиск по реквизитам справочников и полям регистров.
- Наиболее эффективно ставить индексы на поля, используемые в отборе и соединениях.
- Индексы создаются автоматически для ключевых полей, остальные можно настраивать вручную.
Оптимизация запросов повышает скорость обработки данных и производительность системы.
